# Walmart Confidence Interval and CLT

## Project Overview
This project analyzes Walmart customer purchasing behavior using Confidence Interval and Central Limit Theorem concepts in Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Files Included
- walmart_analysis.ipynb
- Walmart_Business_Case.pdf

## Analysis Performed
- Descriptive Statistics
- Confidence Interval Analysis
- Central Limit Theorem (CLT)
- Customer Purchase Analysis
- Data Visualization

## Key Insights
- Spending behavior analysis
- Gender-based purchase trends
- Confidence interval estimation
- Sampling distribution analysis
